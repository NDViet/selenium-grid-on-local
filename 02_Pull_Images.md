## Run below docker commands to pre-pull the required images

```sh
docker pull selenium/base:latest
docker pull selenium/distributor:latest
docker pull selenium/event-bus:latest
docker pull selenium/hub:latest
docker pull selenium/node-base:latest
docker pull --platform linux/amd64 selenium/node-chrome:latest
docker pull selenium/node-chromium:latest
docker pull --platform linux/amd64 selenium/node-edge:latest
docker pull selenium/node-firefox:latest
docker pull selenium/node-docker:latest
docker pull selenium/router:latest
docker pull selenium/sessions:latest
docker pull selenium/session-queue:latest
docker pull --platform linux/amd64 selenium/standalone-chrome:latest
docker pull selenium/standalone-chromium:latest
docker pull --platform linux/amd64 selenium/standalone-edge:latest
docker pull selenium/standalone-firefox:latest
docker pull selenium/standalone-docker:latest
docker pull selenium/video:latest
```
